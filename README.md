# Social Connect

## Resumo

A **Social Connect** é uma plataforma inovadora focada em conectar pessoas e empresas nas redes sociais, promovendo relacionamentos digitais de forma estratégica e eficiente. O projeto consiste em um site institucional que apresenta a empresa, sua equipe, formas de contato e links para as redes sociais.

## Tecnologias utilizadas

- HTML5
- CSS3 (estilos inline no próprio arquivo)
- GitHub Pages (hospedagem)
- Git / GitHub (versionamento e colaboração)
